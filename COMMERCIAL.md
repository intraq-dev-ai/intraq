# Commercial Boundary

This source tree is designed to be useful on its own while preserving a paid
product. Forking, modification, evaluation, and internal self-hosting are
expected under the license.

Paid hosting, white-label resale, OEM redistribution, paid third-party
implementation/operations, or use in a competing commercial analytics, BI,
dashboard, SQL-assistant, or AI-reporting service requires an IntraQ commercial
agreement.

## Included

- Dashboards, Dashboard Builder, Analyzer, SQL Editor, data sources, data
  dictionary, public dashboard sharing, SMTP, and MCP access.
- Bring-your-own AI provider configuration.
- Metadata-aware AI grounded in table names, field names, data dictionary
  entries, saved SQL models, relationships, dashboard context, and safe
  sample/result summaries.
- MCP tools for dashboards, Analyzer, data sources, SQL models, and product API
  access.

## Paid Product Keeps

- Advanced AI Studio and proprietary business-language intelligence.
- Domain packs, feedback learning, eval workflows, and managed AI governance.
- Multi-tenant control, MFA/RBAC/SSO administration, audit/security controls,
  subscription management, billing, and managed hosting.
- Commercial support, implementation, OEM agreements, and managed-service
  rights.

## AI Rule

AI in this source tree should not be weak. It should build useful dashboards and
Analyzer answers from the customer's own schema and data models. It should not
contain the proprietary AI Studio layer that teaches industry-specific meanings,
policy packs, or learned business rules.
