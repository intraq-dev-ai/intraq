# Public Source Scope

This repository is the generalized intraQ public source tree.

## Included

- self-hosted operational analytics;
- dashboards and dashboard builder workflows;
- Analyzer and SQL assistant workflows grounded in local metadata and models;
- generic data-source management;
- MCP access features suitable for local/self-hosted use;
- generic sample data and seed metadata.

## Excluded

- credentials, provider keys, database passwords, or private env files;
- client-specific data, documents, screenshots, exports, or acceptance evidence;
- private operational and go-to-market material;
- paid AI Studio logic;
- proprietary domain intelligence packs;
- feedback learning loops and eval pipelines;
- multi-tenant governance, MFA/RBAC, billing, managed-service code, and control
  plane tooling.

## Contribution boundary

Contributions should improve the generalized product. If a change is only useful
for one customer, one private deployment, or one paid managed-service workflow,
keep it outside this repository.
